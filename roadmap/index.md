- Security terminology

## Phase 2 — Practical Skills
- Enumeration
- Burp Suite
- Log analysis
- OSINT

## Phase 3 — Advanced Topics
- Privilege escalation
- Red teaming
- Pentesting workflow
- Active Directory
- Detection and response
