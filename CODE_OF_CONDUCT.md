# Code of Conduct

## Our Pledge

We want this community to be welcoming, respectful, and useful for everyone.

## Expected Behavior

- Be respectful to others
- Stay on topic
- Help beginners
- Share useful and ethical content
- Use clear and kind language
- Respect different backgrounds and skill levels

## Unacceptable Behavior

- Harassment
- Hate speech
- Spam
- Trolling
- Sharing illegal content
- Posting flags, private lab answers, or leaked material
- Disrespecting other members
- Personal attacks

## Community Rules

- Keep discussions ethical and legal
- Be beginner-friendly when possible
- Do not insult people for asking questions
- Do not post malicious content
- Do not break GitHub’s rules or laws

## Enforcement

If someone breaks the rules, maintainers may:

- remove content
- warn the user
- restrict participation
- close discussions or issues if needed

## Reporting Issues

If you see a problem, please report it respectfully through an issue or discussion.

## Scope

This code of conduct applies to:
- the repository
- discussions
- issues
- pull requests
- any community activity connected to this project

## Final Note

We are building a student cybersecurity community.  
Respect and ethics matter here.
